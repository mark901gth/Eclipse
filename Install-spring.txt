To install the Spring Tool Set in eclipse:

Menu: Help / Install New Software / [Add]

	Name:		Spring Tool Suite
	Location:	http://dist.springsource.com/release/TOOLS/update/e4.3/

Select:
	Core / Spring IDE
  	Extensions / Spring IDE
 *	Integrations / Spring IDE
  	Resources / Spring IDE
